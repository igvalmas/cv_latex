# 📄 Ignacio Valmaseda - CV Repository

Este repositorio contiene mi ecosistema de currículums optimizados para diferentes mercados y objetivos profesionales, desde perfiles altamente cuantitativos hasta consultoría de negocio financiero.

## 🏗️ Estructura del Proyecto

El repositorio está organizado en tres enfoques estratégicos:

* **/cv_standard**: Perfil enfocado a **Quantitative Analytics & Data Science**. 
    * *Idioma:* Inglés.
    * *Objetivo:* Multinacionales, puestos remotos Tech y posiciones de Data Scientist.
* **/cv_finance**: Perfil de **"High-End" Finance & Profitability Architect**.
    * *Idioma:* Inglés.
    * *Objetivo:* Fondos de inversión, banca de inversión, consultoras de élite y proyectos Quant complejos.
* **/cv_mundano**: Perfil de **Consultoría de Negocio y Rentabilidad**.
    * *Idioma:* Español.
    * *Objetivo:* Mercado nacional (España/Canarias), PYMEs grandes, consultoría local y emprendimiento.
* **/img**: Recursos gráficos y activos visuales.

## 🚀 Instrucciones de Compilación

Todos los CVs están desarrollados en LaTeX utilizando la clase `moderncv`.

1. Entrar en la carpeta del perfil deseado.
2. Limpiar archivos auxiliares si existen: `del *.aux *.log *.out`.
3. Compilar mediante el comando:
   ```bash
   pdflatex nombre_del_archivo.tex
